# starkow.dev

sources of my [starkow.dev](https://starkow.dev) website.

you will need [node.js](https://nodejs.org/en/) installed.

## installing

##### npm

```sh
npm install
```

##### yarn

```sh
yarn
```

## running

##### npm

```sh
npm run dev
```

##### yarn

```sh
yarn dev
```

## building

##### npm

```sh
npm run build
```

##### yarn

```sh
yarn build
```

all the necessary files will be generated in the `dist` folder.
